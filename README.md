# Hi, Adam!
Here's how to run this on your computer.
## 1. Click the blue `<Code>` button

## 2. Launch in Codespaces
1. Choose the **Codespaces** tab
2. Click **Create codespace on main**
  - In 30-60 seconds, GitHub will set up a cloud dev environment for you.

## 3. Wait for the environment to configure
The project is preconfigured to do the complicated work for you, like downloading things.

## 4. Open the Notebook
In the left sidebar:
1. Navigate to the `notebooks` folder
2. Open `Essentials.ipynb`
